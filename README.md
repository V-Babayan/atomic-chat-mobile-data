# atomic-chat-mobile-data

On-device model catalog for [`atomic-chat-mobile-app`](https://github.com/V-Babayan/atomic-chat-mobile-app).

Edited via the admin panel at https://v-babayan.github.io/atomic-chat-admin/ (writes go through the GitHub Contents API with a fine-grained PAT).

Consumed by the mobile app on startup via jsDelivr:

```
https://cdn.jsdelivr.net/gh/V-Babayan/atomic-chat-mobile-data@main/models.json
```
